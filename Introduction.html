Internet Working: Device Communication

The basic idea

Explains how two computers or devices communicate and send data to each other.

The process

Communication is about sending and receiving messages between a sender and a receiver.

Data Transfer Mechanism

What's needed to send a message

Three essentials:

  1. Sender's address
  2. Receiver's address
  3. The message (payload)

How it works

The sender "boxes up" (encodes) the message and the receiver "unboxes" (decodes) it on arrival.

Introduction to IP Addresses

What an IP address is

An IP address is the numeric address a device uses to communicate on the internet.

Is it permanent?

No — IP addresses are typically assigned by a network when a device connects and released when it disconnects (dynamic allocation). Some devices or services may have static (permanent) IPs.
