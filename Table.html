HTML Tables — Notes

Understanding how to arrange data neatly in rows and columns.

Why use tables?

Tables are used to arrange data neatly in rows and columns, just like a spreadsheet.

They are useful for showing structured information like marks, schedules, etc.

Key Tags

  • table → Main container for the table.
  • tr → Table Row. Each row in the table.
  • th → Table Header. Used for headings like "Name" or "Marks".
  • td → Table Data. Used for the actual values inside cells.

Creating a Basic Table

  1. Start with table tag.
  2. Add a row for headers using tr and th.
  3. Add more rows using tr and td.

Example Table:

Name Marks
Aryan 85
Raj 90

Advanced Stuff

  • rowspan → makes a cell span across multiple rows.
  • colspan → makes a cell span across multiple columns.
