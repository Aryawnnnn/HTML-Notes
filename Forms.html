HTML Form Tutorial Notes

Notes about creating and working with forms in HTML.

I. Introduction to Forms

Forms are used to collect user input like name, email, or feedback.

They are essential for login, signup, surveys, and search bars.

II. Creating a Form

  • form → container that holds all input fields.
  • action → defines where the data will be sent.
  • method → GET (data in URL) or POST (data hidden, safer).

III. Input Types & Attributes

  • text → for names or general text.
  • email → validates proper email format.
  • password → masks characters.
  • number → restricts input to numbers.
  • radio → select one option.
  • checkbox → select multiple options.
  • date → pick a date.
  • submit → button to send form data.

IV. Important Attributes

  • id → unique identifier for an element.
  • for → connects a label to an input (must match the input’s id).
  • name → key used when data is sent to the server.
  • placeholder → hint text inside an input field.
  • required → makes the field mandatory.
  • value → pre-fills the input with default text.

V. Textarea

The textarea tag allows multi-line input, useful for comments or messages.
