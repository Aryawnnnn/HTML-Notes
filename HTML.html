HTML Notes: Introduction & Fundamentals

What is HTML?

HTML stands for HyperText Markup Language.

It is the foundation of every webpage, providing the structure and content.

"HyperText" refers to links that connect different web pages.

Getting Started: Tools & Setup

  • Use VS Code (Visual Studio Code) as the editor.
  • Useful extensions:
    • Live Server – shows live updates in the browser
    • Live Preview – shows a preview inside VS Code
    • Prettier – automatically formats your code
  • All HTML files must end with .html (for example: index.html)

Key HTML Elements

Headings(<"h1 /h2..h6">)

  • Headings are used for titles and subtitles.
  • They range from level 1 (main heading) to level 6 (least important).
  • They are important for page structure and SEO, not just for size.

Paragraphs(<"p">)

Paragraphs are used for normal text content.

Line Break and Horizontal Rule(<"hr">)

  • Line Break: Moves content to the next line
  • Horizontal Rule: Draws a horizontal line to separate sections

Lists(<"ul /ol">)

  • Unordered list: Creates a bulleted list
  • Ordered list: Creates a numbered list

Nested Lists

A nested list means placing one list inside another.

Links(<"a href=">)

Links are used to connect one page to another or to external websites.

  • href: defines the web address
  • target=" _blank": opens the link in a new tab

Images(<"img src">)

Images can be added to a webpage. The source (src) defines where the image comes from, and the alternative text (alt) describes the image for accessibility or when it fails to load.
