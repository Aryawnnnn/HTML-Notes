HTML Media Tutorial Notes

Notes about using images, audio, video, and embedded content in HTML.

I. Introduction to Media

Media elements (images, audio, video, iframe) allow us to add rich content to web pages.

They improve user experience by including visuals, sound, and interactive content.

II. Image Tag

  • <img> → used to display images.
  • src → source of the image.
  • alt → alternate text (shown if image is missing, important for accessibility).
  • width / height → size of the image.

III. Audio Tag

  • <audio> → embeds audio files.
  • controls → adds play, pause, and volume buttons.
  • autoplay → plays automatically when page loads.
  • loop → repeats audio automatically.
  • muted → starts audio in muted state.

IV. Video Tag

  • <video> → embeds video files.
  • controls → adds play, pause, and volume controls.
  • autoplay → video starts automatically.
  • loop → repeats video automatically.
  • muted → starts muted.
  • poster → thumbnail shown before video plays.

V. Embedding External Media (iframe)

  • <iframe> → used to embed external content like YouTube videos, Google Maps, or other websites.
  • src → the URL of the embedded content.
  • width / height → size of the iframe.
  • allowfullscreen → lets video go full screen.
